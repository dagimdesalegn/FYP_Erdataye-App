# Security Policy

## Supported Versions

This project follows a branch-based support model.

| Branch | Supported |
| ------ | --------- |
| main | :white_check_mark: |
| abeni | :white_check_mark: |
| dependabot/npm_and_yarn/eslint-10.2.1 | :white_check_mark: |
| other branches | :x: |

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Use GitHub Security Advisories private reporting:
1. Open the repository on GitHub.
2. Go to Security > Advisories.
3. Click Report a vulnerability.
4. Include steps to reproduce, impact, and suggested fix.

Response targets:
- Initial acknowledgement: within 72 hours.
- Triage decision: within 7 days.
- Status updates: every 7 days until resolved.

If accepted, we will patch main first and then backport if needed.
If declined, we will provide a short explanation in the advisory thread.
